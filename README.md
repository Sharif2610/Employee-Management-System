🏢 Employee Management System

A full-stack Employee Management System built using Python, Django, React.js, and MySQL.
This project helps organizations manage employee records efficiently with secure authentication and role-based access control.

✨ Features
🔐 Secure user authentication using JWT
👨‍💼 Role-based access control (Admin / Employee)
➕ Add new employee records
✏️ Update employee details
❌ Delete employee records
📋 View employee list with details
🔍 Search and filter employees
📱 Responsive UI using React.js
🔗 RESTful APIs for smooth frontend-backend communication
🛠️ Tech Stack

Frontend: React.js, HTML5, CSS3, JavaScript
Backend: Python, Django, Django REST Framework
Database: MySQL
Authentication: JWT (JSON Web Token)
Tools: Git, GitHub, Postman, VS Code
